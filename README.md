# Retos AI Native — Paola Peralta

Repositorio de proyectos y retos del programa AI Native.

## Proyectos

| Carpeta | Descripción |
|---|---|
| `Goal_Navigator/` | App OKR completa construida con React + Supabase |
| `Reto3_Resumen_Ejecutivo/` | Dashboard HTML de resumen ejecutivo semanal |
| `Reto_n8n_Cumpleanos/` | Flujo n8n: Google Sheets → Google Slides automático |
| `Reto_n8n_Referidores/` | Flujo n8n: webhook → Gmail notificaciones referidores |
| `Reto_Analisis_Energia/` | Scripts Python análisis y pronóstico de consumo eléctrico |
| `Reto_AppScript_Visitantes/` | Apps Script: notificación automática de visitantes a Slack |

## Tecnologías usadas
- React + TypeScript + Supabase
- Python (pandas, numpy)
- n8n (automatización de flujos)
- Google Apps Script
- HTML / CSS
